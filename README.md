# flexbog.github.io
Projeto criado para praticar CSS Flexbox, explorando propriedades como display: flex, justify-content, align-items e flex-direction, com o objetivo de construir layouts organizados e responsivos utilizando HTML5 e CSS3.

# Projeto Flexbox

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conceitos de **CSS Flexbox**.

## O que foi praticado
- display: flex
- justify-content
- align-items
- Flex-grow
- Flex-basis
- Flex Container
- flex-direction
- Centralização de elementos
- Organização de layout


## Tecnologias utilizadas
- HTML5
- CSS3

## Objetivo
Aprimorar habilidades em layout e posicionamento de elementos, com foco no desenvolvimento front-end.
